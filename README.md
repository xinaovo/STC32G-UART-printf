# STC32G UART printf Project Template

## Setup & Build

- Install and configure Keil C251

- Install STC MCU Database via AiCube-ISP (formerly AIAPP-ISP, STC-ISP)

- Patch Keil using "Keil Interrupt Extension Plugin", provided by STC

- Clone this project and build with Keil

## Notes

The encoding of the source code is GB2312. Chinese characters may not be displayed properly if incorrect encoding settings are applied.
